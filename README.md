基于Selenium的前途无忧网职位信息爬取 
=======
## 开发环境
系统:windows <br>
IDE:Pycharm <br>
Web框架:Django <br>

## 项目

### 项目描述
本项目使用Selenium框架来实现网页爬虫功能,利用re模块来进行匹配数据,再使用多进程来提高爬虫效率,并把数据存入MongoDB中        <br>

### 项目功能
*  对前途无忧网站的职位信息进行爬取
*  将职位的各种数据存入数据库中


## 总结
本项目有许多可改进以及优化的地方  <br>
使用分布式爬虫<br>
可将数据保存为各种文件<br>
